## 二、基础数据配置

### 2.1 级号管理

**路径**：德育配置 → 级号管理

**功能说明**：
- 级号代表入学年份，如"2023级"、"2024级"
- 级号与班级、学生关联，是德育评价的基础

**操作步骤**：

1. **查看级号列表**
   - 进入级号管理页面，查看现有级号
   - 系统已从 Excel 模板自动导入级号数据

2. **新增级号**
   - 点击"新增级号"按钮
   - 填写级号名称（如：2025级）
   - 填写入学年份（如：2025）
   - 点击确定保存

3. **删除级号**
   - 仅可删除无关联班级的级号
   - 若提示"存在关联班级"，需先删除相关班级

**字段说明**：

| 字段 | 说明 | 示例 |
|------|------|------|
| 级号名称 | 显示名称 | 2023级 |
| 入学年份 | 用于计算年级 | 2023 |

---

### 2.2 班级管理

**路径**：德育配置 → 班级管理

**功能说明**：
- 管理班级基本信息
- 班级与级号、学生关联

**操作步骤**：

1. **查看班级列表**
   - 支持按级号筛选
   - 显示班级人数统计

2. **新增班级**
   - 点击"新增班级"按钮
   - 选择所属级号
   - 填写班级编码（如：202301）
   - 填写班级号（如：1）
   - 填写班级名称（如：高一1班）
   - 填写班主任姓名
   - 点击确定保存

3. **编辑班级**
   - 点击操作列"编辑"按钮
   - 修改班级信息
   - 点击确定保存

4. **删除班级**
   - 仅可删除无学生的班级
   - 若提示"存在学生"，需先处理学生数据

**字段说明**：

| 字段 | 说明 | 示例 |
|------|------|------|
| 班级编码 | 唯一标识 | 202301 |
| 所属级号 | 关联级号 | 2023级 |
| 班级号 | 班级序号 | 1 |
| 班级名称 | 显示名称 | 高一1班 |
| 班主任 | 班主任姓名 | 张老师 |

---

### 2.3 学生管理

**路径**：德育配置 → 学生管理

**功能说明**：
- 管理学生基本信息
- 学生与班级、级号关联
- 支持学籍变动管理

**操作步骤**：

1. **查看学生列表**
   - 支持按级号、班级筛选
   - 支持按姓名、学号搜索
   - 分页显示，默认每页20条

2. **新增学生**
   - 点击"新增学生"按钮
   - 填写学号（唯一标识）
   - 填写姓名
   - 选择性别
   - 选择所属班级
   - 填写出生日期（可选）
   - 点击确定保存

3. **编辑学生**
   - 点击操作列"编辑"按钮
   - 修改学生信息

4. **更新学生状态**
   - 点击操作列"状态"按钮
   - 选择新状态：在校/毕业/转出/休学
   - 填写变动原因
   - 点击确定保存

**字段说明**：

| 字段 | 说明 | 示例 |
|------|------|------|
| 学号 | 唯一标识 | 20230101 |
| 姓名 | 学生姓名 | 张三 |
| 性别 | 男/女 | 男 |
| 所属班级 | 关联班级 | 高一1班 |
| 状态 | 学籍状态 | 在校 |
| 出生日期 | 生日信息 | 2008-05-15 |

## 三、学年学期配置

### 3.1 创建学年

**路径**：德育配置 → 学年学期

**操作步骤**：

1. 在"学年管理"区域点击"新增学年"
2. 填写学年名称（如：2025-2026学年）
3. 选择开始日期（默认9月1日）
4. 选择结束日期（默认次年7月15日）
5. 点击确定保存

### 3.2 创建学期

**操作步骤**：

1. 在"学期管理"区域点击"新增学期"
2. 选择所属学年
3. 填写学期名称（如：2025-2026上）
4. 选择开始日期
   - 上学期：9月1日
   - 下学期：次年2月20日
5. 选择结束日期
   - 上学期：次年1月20日
   - 下学期：次年7月15日
6. 点击确定保存

### 3.3 设置当前学期

**操作步骤**：

1. 在学期列表中找到目标学期
2. 点击操作列"设为当前"按钮
3. 确认操作
4. 系统自动将其他学期设为非当前

**注意**：
- 只能有一个当前学期
- 德育评价按当前学期计算

## 四、事件类型配置

### 4.1 日常事件类型

**路径**：德育配置 → 事件类型 → 日常事件类型

**功能说明**：
- 定义日常表现事件类型
- 分为积极事件（加分）和消极事件（扣分）

**操作步骤**：

1. **查看事件类型**
   - 点击"全部/积极事件/消极事件"筛选
   - 查看事件名称、分值、状态

2. **新增事件类型**
   - 点击"新增事件类型"按钮
   - 填写事件名称（如：拾金不昧）
   - 选择事件类型（积极/消极）
   - 填写分值（自动设置正负）
   - 填写描述（可选）
   - 点击确定保存

3. **编辑事件类型**
   - 点击操作列"编辑"按钮
   - 修改事件信息

4. **启用/禁用事件类型**
   - 点击操作列"禁用/启用"按钮
   - 禁用后该类型不再出现在记录选择中

### 4.2 校级事件类型

**路径**：德育配置 → 事件类型 → 校级事件类型

**功能说明**：
- 定义校级荣誉和违纪事件
- 按级别区分（国家级/省级/市级/校级）

**操作步骤**：

1. **新增校级事件**
   - 点击"新增事件类型"按钮
   - 填写事件名称（如：三好学生）
   - 选择事件类型（荣誉奖励/违纪处分）
   - 选择事件级别（国家级/省级/市级/校级）
   - 填写分值
   - 填写描述
   - 点击确定保存

### 4.3 模板下载与批量导入

**模板下载**：

1. 进入事件类型管理页面
2. 点击"下载模板"按钮
3. 选择模板类型：
   - 日常事件类型模板
   - 校级事件类型模板
4. 保存CSV文件

**模板格式**：

日常事件类型模板 (`daily_events.csv`)：
```csv
事件名称,事件类型,分值,描述
拾金不昧,积极,3,主动上交拾得物品
课堂积极发言,积极,2,课堂主动回答问题
迟到,消极,-2,上课铃响后进入教室
```

校级事件类型模板 (`school_events.csv`)：
```csv
事件名称,事件类型,事件级别,分值,描述
三好学生,荣誉奖励,校级,10,校级三好学生称号
优秀班干部,荣誉奖励,校级,8,校级优秀班干部
打架斗殴,违纪处分,校级,-10,校内打架行为
```

**批量导入**：

1. 按模板格式填写数据
2. 点击"批量导入"按钮
3. 选择CSV文件
4. 系统验证数据格式
5. 显示导入预览
6. 确认导入
7. 查看导入结果

## 五、德育任务配置

**路径**：德育评价 → 德育任务

**功能说明**：
- 定义各年级德育任务
- 学生完成任务获得积分

**操作步骤**：

1. **查看任务列表**
   - 按级号筛选任务
   - 查看任务名称、分值、截止类型

2. **新增德育任务**
   - 点击"新增任务"按钮
   - 选择适用级号
   - 填写任务名称（如：志愿服务）
   - 填写任务描述
   - 填写完成得分
   - 选择截止类型：
     - 学期截止：本学期内完成
     - 学年截止：本学年完成
     - 无截止：随时可完成
   - 选择是否必修
   - 点击确定保存

**模板导入**：

德育任务模板 (`moral_tasks.csv`)：
```csv
级号名称,任务名称,任务描述,分值,截止类型,是否必修
2023级,志愿服务,参加志愿服务活动,5,学年,是
2023级,社会实践,参加社会实践活动,5,学年,否
2024级,读书笔记,提交读书笔记,3,学期,是
```

## 七、常见问题

### Q1: 如何批量导入学生数据？

学生数据通过 Excel 模板导入：
1. 准备 `checkTemplate.xlsx` 文件
2. 在学生工作表填写数据
3. 运行数据迁移脚本：`python scripts/migrate_to_mysql.py`
4. 验证导入结果

### Q2: 如何修改已毕业学生的状态？

1. 进入学生管理页面
2. 搜索该学生
3. 点击"状态"按钮
4. 选择"毕业"状态
5. 填写变动原因
6. 保存

### Q3: 学期切换后，德育评价数据如何处理？

系统自动按学期隔离数据：
- 日常表现记录关联到记录时的学期
- 德育评价按学期计算
- 学期末可生成评价汇总

### Q4: 事件类型可以修改分值吗？

可以修改：
1. 进入事件类型管理
2. 编辑目标事件类型
3. 修改分值
4. 保存

**注意**：修改分值只影响后续记录，已存在记录的分值不变。

### Q5: 如何删除错误导入的事件类型？

1. 先禁用该事件类型
2. 确认无关联记录后
3. 可通过数据库直接删除（需要管理员权限）
