## 通用设计说明

### Prompt 层级结构

```
每类问题 = System Prompt + 首次分析 Prompt + 追问 Prompt 组
                                              ├── 信息补充追问
                                              ├── 风险深探追问
                                              ├── 干预方案追问
                                              ├── 进展评估追问
                                              └── 结案总结追问
```

### 变量约定

所有 Prompt 中使用 `{{变量名}}` 标记动态填充字段：

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{student_name}}` | 学生姓名 | 张三 |
| `{{grade_class}}` | 年级班级 | 高一3班 |
| `{{gender}}` | 性别 | 男/女 |
| `{{description}}` | 问题描述原文 | 教师填写内容 |
| `{{priority}}` | 优先级 | 普通/紧急/非常紧急 |
| `{{history_messages}}` | 历史对话记录 | JSON 格式对话历史 |
| `{{last_analysis}}` | 上次AI分析摘要 | 文本摘要 |
| `{{days_elapsed}}` | 诊疗已持续天数 | 数字 |
| `{{teacher_new_input}}` | 教师最新补充信息 | 文本 |

### 追问触发机制

```
触发条件判断树：

教师发送新消息
    ├── 包含新情况描述 → 触发「信息补充追问」
    ├── 包含风险信号词（哭泣/逃跑/威胁等）→ 触发「风险深探追问」
    ├── 询问"怎么办/如何处理" → 触发「干预方案追问」
    ├── 反馈已干预结果 → 触发「进展评估追问」
    └── 请求关闭会话 → 触发「结案总结追问」
```

## 一、心理问题完整方案

### 1.1 System Prompt

```
【角色定义】
你是一位受过专业训练的学生心理健康顾问AI，协助学校教育工作者识别、评估并应对学生心理健康问题。你的分析基于认知行为理论、生态系统理论和危机干预原则。

【核心原则】
1. 安全第一：任何涉及自伤、自杀、他伤的信号必须在分析最前端标注【高风险预警】，并给出立即行动步骤
2. 不替代诊断：提供教育干预建议，不做临床诊断，不使用"抑郁症""焦虑障碍"等临床标签
3. 去标签化：使用描述性语言，如"情绪持续低落超过两周""明显的社交回避表现"
4. 保密原则：提醒教师注意信息保密，仅在安全风险时打破保密
5. 发展视角：以青少年正常心理发展阶段为参照，避免病理化正常波动

【输出格式要求】
- 使用 Markdown 结构化输出
- 高风险内容用 ⚠️ 标注，放在最前
- 建议具体可操作，避免泛化表述如"多关注"
- 每次追问必须引用上轮关键信息，保持分析连贯性
```

## 二、行为问题完整方案

### 2.1 System Prompt

```
【角色定义】
你是一位专注于学生行为干预的教育顾问AI，分析框架基于行为功能分析（FBA）和正向行为支持（PBS）理论。

【核心原则】
1. 行为是沟通：每个问题行为背后都有其功能，是学生表达未满足需求的方式
2. 就事论事：评估行为本身，不评判人格和品德
3. 正向优先：正向行为支持优先于惩罚措施
4. 数据导向：基于可观察、可测量的行为描述，避免模糊归因
5. 家校一致：有效干预需要家校在规则和强化上保持一致

【输出格式要求】
- 行为描述必须操作性定义（可观察、可测量）
- ABC 分析必须基于教师描述的具体情境
- 干预措施分为立即可用和需要准备两类
- 追问时必须明确引用行为频率/强度等数据
```

```python
def select_followup_prompt(consultation_type, teacher_message, history, days_elapsed):
    """
    根据教师消息内容和诊疗阶段，选择最合适的追问 Prompt
    """
    triggers = get_triggers(consultation_type)  # 获取对应类型的触发规则

    # 优先级1：风险信号 → 立即触发风险深探
    if any(word in teacher_message for word in triggers.get("风险深探", [])):
        return "risk_exploration_prompt"

    # 优先级2：行为升级
    if any(word in teacher_message for word in triggers.get("行为升级", [])):
        return "escalation_prompt"

    # 优先级3：结案请求
    if any(word in teacher_message for word in triggers.get("结案总结", [])):
        return "closure_prompt"

    # 优先级4：进展反馈
    if any(word in teacher_message for word in triggers.get("进展评估", [])):
        if days_elapsed >= 14:  # 至少2周才建议结案评估
            return "progress_evaluation_prompt"

    # 优先级5：具体帮助请求
    for prompt_type, keywords in triggers.items():
        if any(word in teacher_message for word in keywords):
            return f"{prompt_type}_prompt"

    # 默认：信息补充追问
    return "information_supplement_prompt"
```

### 7.3 多轮对话历史管理

```python
# 建议的历史消息管理策略
def build_conversation_history(messages, max_tokens=6000):
    """
    保留最近N条消息，超出时保留：
    1. 第一条（首次分析，含完整背景）
    2. 所有包含【高风险预警】的消息
    3. 最近3条消息
    """
    result = []
    # 始终保留首条
    result.append(messages[0])
    # 保留风险相关消息
    risk_msgs = [m for m in messages[1:-3] if "高风险" in m.get("content", "")]
    result.extend(risk_msgs)
    # 保留最近3条
    result.extend(messages[-3:])
    return result
```

```python
# lesson/models/datas_api/moral/consultation_ai.py

import json
from datetime import datetime
from typing import Optional, Dict, List
from openai import OpenAI
from config.config import Config

from .consultation_prompts import (
    SYSTEM_PROMPTS,
    FIRST_ANALYSIS_PROMPTS,
    FOLLOWUP_PROMPTS,
    TRIGGER_KEYWORDS

def call_ai_model(system_prompt: str, user_prompt: str, temperature: float = 0.3) -> str:
    """
    调用阿里百炼 Kimi K2.5 模型
    
    Args:
        system_prompt: 系统提示词（角色定义）
        user_prompt: 用户提示词（具体问题）
        temperature: 温度参数，越低越稳定
    
    Returns:
        AI 生成的回复内容
    """
    try:
        client = OpenAI(
            api_key=Config().get_config("bailian_token", "token.yaml"),
            base_url="https://coding.dashscope.aliyuncs.com/v1",
            timeout=30,
        )
        completion = client.chat.completions.create(
            model="kimi-k2.5",
            messages=[
                {"role": "system", "content": system_prompt},
                {"role": "user", "content": user_prompt},
            ],
            temperature=temperature,
            max_tokens=2500,
        )
        return completion.choices[0].message.content.strip()
    except Exception as e:
        return f"AI 分析暂时不可用，请稍后重试。错误：{str(e)}"


def fill_prompt_template(template: str, variables: Dict) -> str:
    """
    填充 Prompt 模板变量
    
    Args:
        template: 包含 {{变量名}} 的模板字符串
        variables: 变量字典
    
    Returns:
        填充后的 Prompt
    """
    result = template
    for key, value in variables.items():
        placeholder = "{{" + key + "}}"
        result = result.replace(placeholder, str(value) if value else "暂无")
    return result


def generate_initial_analysis(
    consultation_type: str,
    student: Dict,
    description: str,
    priority: str = "normal"
) -> Dict:
    """
    生成首次 AI 分析
    
    Args:
        consultation_type: 诊疗类型（psychological/behavior/academic/comprehensive）
        student: 学生信息字典（包含 name, class_name, gender 等）
        description: 问题描述
        priority: 优先级
    
    Returns:
        包含 analysis, suggestions, risk_assessment 的字典
    """
    # 获取 System Prompt
    system_prompt = SYSTEM_PROMPTS.get(consultation_type, SYSTEM_PROMPTS["comprehensive"])
    
    # 构建变量
    variables = {
        "student_name": student.get("name", ""),
        "grade_class": student.get("class_name", ""),
        "gender": student.get("gender", "未知"),
        "description": description or "暂无详细描述",
        "priority": priority or "普通",
    }
    
    # 获取并填充首次分析 Prompt
    first_prompt_template = FIRST_ANALYSIS_PROMPTS.get(consultation_type, FIRST_ANALYSIS_PROMPTS["comprehensive"])
    user_prompt = fill_prompt_template(first_prompt_template, variables)
    
    # 根据类型调整温度
    temp_map = {
        "psychological": 0.25,  # 心理问题需要最稳定输出
        "behavior": 0.30,
        "academic": 0.30,
        "comprehensive": 0.35,
    }
    temperature = temp_map.get(consultation_type, 0.3)
    
    # 调用 AI
    analysis_content = call_ai_model(system_prompt, user_prompt, temperature)
    
    # 提取风险等级
    risk_assessment = "low"
    if "🔴 高风险" in analysis_content or "高风险预警" in analysis_content:
        risk_assessment = "high"
    elif "🟡 中风险" in analysis_content:
        risk_assessment = "medium"
    
    # 提取建议（简单解析）
    suggestions = []
    lines = analysis_content.split("\n")
    for line in lines:
        if line.strip().startswith(("1.", "2.", "3.", "4.", "5.", "6.")):
            suggestions.append(line.strip())
    
    return {
        "analysis": analysis_content,
        "suggestions": suggestions[:6] if suggestions else ["请查看完整分析报告"],
        "risk_assessment": risk_assessment,
    }


def detect_followup_type(
    consultation_type: str,
    teacher_message: str,
    days_elapsed: int = 0
) -> str:
    """
    根据教师消息内容检测追问类型
    
    Args:
        consultation_type: 诊疗类型
        teacher_message: 教师发送的消息内容
        days_elapsed: 诊疗已持续天数
    
    Returns:
        追问类型（risk_exploration/information_supplement/intervention_deepen 等）
    """
    triggers = TRIGGER_KEYWORDS.get(consultation_type, {})
    
    # 优先级1：风险信号 → 立即触发风险深探
    risk_keywords = triggers.get("risk_exploration", [])
    for keyword in risk_keywords:
        if keyword in teacher_message:
            return "risk_exploration"
    
    # 优先级2：升级信号（行为问题）
    escalation_keywords = triggers.get("escalation", [])
    for keyword in escalation_keywords:
        if keyword in teacher_message:
            return "escalation"
    
    # 优先级3：结案请求
    closure_keywords = triggers.get("closure_summary", [])
    for keyword in closure_keywords:
        if keyword in teacher_message:
            return "closure_summary"
    
    # 优先级4：进展反馈（至少2周才建议结案评估）
    progress_keywords = triggers.get("progress_evaluation", [])
    for keyword in progress_keywords:
        if keyword in teacher_message and days_elapsed >= 14:
            return "progress_evaluation"
    
    # 优先级5：干预请求
    intervention_keywords = triggers.get("intervention_deepen", [])
    for keyword in intervention_keywords:
        if keyword in teacher_message:
            return "intervention_deepen"
    
    # 优先级6：动力重建（学业问题）
    motivation_keywords = triggers.get("motivation_rebuild", [])
    for keyword in motivation_keywords:
        if keyword in teacher_message:
            return "motivation_rebuild"
    
    # 优先级7：家校协同
    family_keywords = triggers.get("family_cooperation", [])
    for keyword in family_keywords:
        if keyword in teacher_message:
            return "family_cooperation"
    
    # 默认：信息补充追问
    return "information_supplement"


def generate_followup_analysis(
    consultation_type: str,
    followup_type: str,
    student: Dict,
    teacher_message: str,
    last_analysis: str = "",
    days_elapsed: int = 0,
    current_risk_level: str = "low"
) -> str:
    """
    生成追问分析
    
    Args:
        consultation_type: 诊疗类型
        followup_type: 追问类型
        student: 学生信息
        teacher_message: 教师新消息
        last_analysis: 上次 AI 分析摘要
        days_elapsed: 诊疗已持续天数
        current_risk_level: 当前风险等级
    
    Returns:
        AI 回复内容
    """
    # 获取 System Prompt
    system_prompt = SYSTEM_PROMPTS.get(consultation_type, SYSTEM_PROMPTS["comprehensive"])
    
    # 获取追问 Prompt 模板
    followup_templates = FOLLOWUP_PROMPTS.get(consultation_type, {})
    prompt_template = followup_templates.get(followup_type, followup_templates.get("information_supplement", ""))
    
    # 检测风险关键词
    risk_keywords = []
    triggers = TRIGGER_KEYWORDS.get(consultation_type, {})
    for keyword in triggers.get("risk_exploration", []):
        if keyword in teacher_message:
            risk_keywords.append(keyword)
    
    # 构建变量
    variables = {
        "student_name": student.get("name", ""),
        "grade_class": student.get("class_name", ""),
        "gender": student.get("gender", "未知"),
        "teacher_new_input": teacher_message,
        "last_analysis": last_analysis[:500] if last_analysis else "暂无",
        "days_elapsed": days_elapsed,
        "current_risk_level": current_risk_level,
        "risk_keywords": ", ".join(risk_keywords) if risk_keywords else "无",
        "initial_risk_level": current_risk_level,  # 假设初始就是当前
    }
    
    # 填充 Prompt
    user_prompt = fill_prompt_template(prompt_template, variables)
    
    # 调用 AI
    temperature = 0.25 if followup_type == "risk_exploration" else 0.3
    return call_ai_model(system_prompt, user_prompt, temperature)


def generate_closure_report(
    consultation_type: str,
    student: Dict,
    start_date: str,
    end_date: str,
    outcome_description: str,
    initial_risk_level: str = "low",
    final_risk_level: str = "low"
) -> str:
    """
    生成结案报告
    
    Args:
        consultation_type: 诊疗类型
        student: 学生信息
        start_date: 开始日期
        end_date: 结束日期
        outcome_description: 教师填写的处理结果
        initial_risk_level: 初始风险等级
        final_risk_level: 最终风险等级
    
    Returns:
        结案报告内容
    """
    # 计算持续天数
    try:
        start = datetime.strptime(start_date, "%Y-%m-%d")
        end = datetime.strptime(end_date, "%Y-%m-%d")
        days_elapsed = (end - start).days
    except:
        days_elapsed = 0
    
    # 获取 System Prompt 和结案模板
    system_prompt = SYSTEM_PROMPTS.get(consultation_type, SYSTEM_PROMPTS["comprehensive"])
    followup_templates = FOLLOWUP_PROMPTS.get(consultation_type, {})
    prompt_template = followup_templates.get("closure_summary", "")
    
    # 构建变量
    variables = {
        "student_name": student.get("name", ""),
        "grade_class": student.get("class_name", ""),
        "gender": student.get("gender", "未知"),
        "start_date": start_date,
        "end_date": end_date,
        "days_elapsed": days_elapsed,
        "outcome_description": outcome_description,
        "initial_risk_level": initial_risk_level,
        "current_risk_level": final_risk_level,
    }
    
    # 填充 Prompt
    user_prompt = fill_prompt_template(prompt_template, variables)
    
    # 调用 AI
    return call_ai_model(system_prompt, user_prompt, 0.3)
```
