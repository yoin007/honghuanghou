# 德育评价系统合并实施计划

## 一、项目概览

### 1.1 项目目标

将现有班级管理系统（Excel + SQLite）与德育评价系统设计方案合并，实现：
- 数据统一管理（Excel → MySQL）
- 角色权限体系统一（7角色RBAC）
- 新增德育评价功能
- 保持现有功能不变

### 1.2 项目范围

| 模块 | 内容 |
|-----|------|
| 数据库 | 37张表、63个外键、2个CHECK约束 |
| 后端API | 103个端点（保留15个，修改5个，新增83个） |
| 前端页面 | 保留现有26个页面，新增德育模块页面 |
| 测试 | 266个测试用例已覆盖核心功能 |

### 1.3 项目周期

**预计总工期: 6-8周**
