# 项目开发计划

## 项目概述
班级数据展示系统，包含前端（Vue3 + Element Plus）和后端（FastAPI）两部分。

---

## 一、配置脱敏（已完成）

### 已完成
- [x] 添加敏感配置文件到 `.gitignore`
- [x] 创建 `.env.example` 配置模板
- [x] 使用 `git filter-repo` 清理 Git 历史中的敏感文件
- [x] 敏感文件清单：
  - `lesson/config/token.yaml` - API 令牌、密钥
  - `lesson/config/wechat.yaml` - 微信机器人配置
  - `lesson/config/config.yaml` - 主配置文件
  - `lesson/config/application.yaml` - 应用配置
  - `lesson/config/network.yaml` - 网络配置

### 部署时需配置
```bash
# 1. 复制配置模板
cp lesson/.env.example lesson/.env

# 2. 编辑实际配置
vi lesson/.env

# 3. 复制敏感配置文件（不提交到 Git）
cp lesson/config/token.yaml.bak lesson/config/token.yaml
```

## 八、开发规范

1. **分支策略**: `feature/*` -> `develop` -> `main`
2. **提交规范**: `feat:`, `fix:`, `refactor:`, `docs:`
3. **代码审查**: PR 必须经过 Code Review
4. **测试**: 核心功能需有单元测试
