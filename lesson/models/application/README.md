## 指令
### rank_to_score
#### 参数
    - rank: 位次
    - category: 类型
    - year: 年份
    - type: 综合（defalult）/专业/文化
### score_to_rank
#### 参数
    - score: 分数
    - category: 类型
    - year: 年份
    - type: 综合（defalult）/专业/文化
### toudang
#### 参数
    - category: 类型
    - zy: 专业
    - year: 年份
    - yx: 院校
    - rank: 位次(default 0)
    - fenshu: 分数(default 0)
    - fangxiang: 方向
    - counts: 数量（default 30）
